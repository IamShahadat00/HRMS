# HRMS
Project Title: HRMS Pro - Full-Stack Human Resource Management System
Description:
A comprehensive web-based HRMS designed to automate organizational staff management, financial structures, and monthly payroll processing. This project follows a decoupled N-Tier architecture to ensure scalability and maintainability.
Key Features:
Role-Based Access Control: Secure login system using JWT and AuthGuards.
Employee Management: Full CRUD operations with server-side searching and status tracking.
Salary Configuration: One-to-One relational mapping to manage base pay, allowances, and fixed deductions.
Payroll Engine: Automated monthly payroll generation featuring a progressive tax calculation algorithm and net salary derivation.
Modern UI: Built with Tailwind CSS for responsiveness and SweetAlert2 for interactive user feedback.
Technical Highlights:
Backend: Implemented Repository-style Service Layer to separate business logic from controllers.
Frontend: Utilized Angular Interceptors to manage JWT token injection globally.
Database: Codes-First migrations with EF Core for automated schema synchronization.
Data Integrity: Handled circular JSON references and enforced model validation on both layers.
